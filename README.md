South Africa Social Safety Net Analysis

Project Overview

This project analyzes the expansion and equity impact of South Africaâ€™s social safety net programs using a modern end-to-end data analytics pipeline built entirely with free tools.

The project demonstrates real-world data analyst skills including data cleaning, wrangling, visualization, and executive storytelling.

---

Business Problem

Governments and policymakers need data-driven insights to evaluate whether social protection programs effectively reach vulnerable populations.

This project answers:

- How has national coverage changed over time?
- Which programs deliver the greatest impact?
- Are benefits reaching lower-income households?

5. Dashboard creation (Looker Studio)
6. Executive reporting (PowerPoint)

---

Dataset

The dataset includes:

- Year
- Program Type
- Coverage Percentage
- Wealth Quintile
- Population Group

Programs analyzed:

- Cash Transfers
- School Feeding
- Food Vouchers
- Disability Benefits
- Public Works

---

Key Insights

Social safety nets expanded steadily over time
Highest coverage observed among poorest households
Coverage decreases as wealth increases (effective targeting)
Cash-based programs show strongest impact

---

Recommendations

- Expand high-performing cash transfer programs
- Monitor middle-income coverage gaps
- Implement real-time analytics dashboards
- Introduce predictive planning models
